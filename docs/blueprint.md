# **App Name**: Eternal Valentine

## Core Features:

- Animated Hero Section: Full-screen hero section with a dark purple gradient background, soft red glow, and animated text fade-in.
- Interactive Buttons: Two interactive buttons with neon glow effect: YES (with a smooth pulse animation) and NO (moves away teasingly when tapped or hovered).
- Love Letter Section: Glassmorphism card with blurred background and typing animation text to display a heartfelt message.
- Responsive Image Carousel: Display memorable moments in a responsive image carousel with swipe gestures on mobile and hover zoom with a purple-red glow border on desktop.
- Surprise Animation: A captivating animation with a screen darkening into a red-purple light burst, hearts, and confetti after the user interacts with the ‘YES’ button.
- Share Functionality: Enable users to share the site via a ‘Share with your Valentine’ button, offering a copy link popup and an Instagram-style story preview design for easy sharing.

## Style Guidelines:

- Primary color: Deep purple gradients (#4B145B and #6A1B9A) evoke mystery and love.
- Accent color: Crimson red accents (#B11226 and #D32F2F) to highlight key interactive elements, increasing urgency.
- Highlight: Rose gold is employed as the 'soft glow' for page highlights.
- Headings font: 'Belleza' serif font for a romantic and classy title.
- Body font: 'Inter' sans-serif font ensures clean, readable text.
- Mobile-first layout with smooth transitions, touch-friendly buttons, and optimized assets for fast loading.
- Subtle animations such as animated text fade-ins, button pulses and neon glows, as well as the full screen Valentine's climax to create a viral, romantic aesthetic.
